규칙

기본규칙

  • - VinylC CSS 규칙 기본으로 합니다.

  • - VinylC CSS 규칙과 차이 있는 부분만 설명합니다.

  • - 프로젝트의 규칙과 다를 경우 프로젝트에서 정한 규칙이 우선합니다.

문법

  • - SCSS 문법으로 사용합니다.

Output Style

  • - compressed 으로 합니다.

SASS 그룹핑

  • - 속성과 속성은 줄바꿈합니다.

  • - 자식 속성 ‘Tab’ 키로 한번 들여쓰기합니다.

  • - 선택자와 중괄호 사이는 space로 한번 들여쓰기 합니다.

예)
@mixin tabs_active($color) {
	border-bottom-color:$color;
	color:$color;
	font-weight:bold;
	@content;
}
예)
.bul_dot {
	margin-top:6px;
	padding-left:7px;
	color:$Bull-color;
	line-height:18px;
	&:first-child {
		margin-top:0;
	}
	.bullet {
		float:left;
		width:2px;
		height:2px;
		margin:8px 0 0 -7px;
		background-color:$Bull-color;
	}
}

Sass 주석

  • - 컴파일/컴파일되지 않는 주석을 사용자 판단에 따라 기입합니다.

  • - 수정주석은 // 컴파일되지 않는 주석으로 합니다.
